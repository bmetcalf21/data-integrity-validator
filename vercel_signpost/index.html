What it does

Validates, cleanses, and deduplicates property and event data for real estate foreclosure tracking. Built for production use with comprehensive error handling and detailed violation reporting.

Key Features

  • Schema validation with regex pattern matching
  • Automatic data normalization (whitespace, casing, column names)
  • Foreign key constraint checking
  • Intelligent deduplication (newest record wins)
  • Detailed rejection reporting with violation_reason
  • Statistics: pass rates, lag analysis, postponement tracking
  • Never crashes on malformed data
